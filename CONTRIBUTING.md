# Contributing

We welcome contributions! This repository hosts financial analysis and strategic reports.

## How to Contribute

1. **Fork the repository**
2. **Review the Analysis**: Read the reports in `reports/` or the presentation in `presentation/`.
3. **Propose Updates**:
    -   Update financial models with recent earnings data.
    -   Refine the strategic framework (SWOT, Porter's 5 Forces).
    -   Add comparable company analysis (Comps).
4. **Open a Pull Request**: Describe your changes and the data sources used.

## Standards
- Cite all data sources (10-K, 10-Q, earnings calls).
- maintain professional formatting in documents.
